Hidup dalam percepatan tekhnologi

2014-10-08 |

Iseng saja kemaren saya mencari game untuk OS Ubuntu. Kenapa mencari game di Ubuntu? Karena di masa saya ini, banyak game yang hanya support untuk OS Windows. Saya tidak akan menceritakan panjang lebar dengan detil (hanya untuk tujuan archiving kepada orang di masa depan) mengenai OS di postingan ini.  Tetapi kenapa saya menggunakan sistem operasi linux distro Ubuntu?

Kenapa? Saya switch dari Windows ke Ubuntu karena ada kegiatan development / coding yang harus saya selesaikan di Ubuntu. Ya karena kegiatan yg bersangkutan dengan development..

Sebenarnya, sistem operasi Ubuntu sudah support dengan Steam. Steam mirip dengan all-in-one marketplace untuk multiple OS. Tetapi yang disupport untuk OS Ubuntu sangat sedikit.

Karena sedikit pilihan untuk free-to-play di Steam, saya tidak jadi mencari game di Steam. Saya hanya menginstall Dota2. Game Moba favorit saya. Dan berjalan lancar disini.


free-to-play game di steam, saat saya menulis tulisan ini

Ok, karena saya tidak jadi mencari game lewat Steam, saya menemui postingan (di askubuntu) yang menganjurkan list game yang support di Ubuntu. Salah satu game nya adalah SecondLife

Yang saya kaget adalah, di halaman SecondLife mereka mendukung Oculus Rift. Apa itu Oculus rift? Tidak akan bahas panjang lebar dengan detil disini. Singkatnya, itu adalah hardware untuk virtual reality. Jadi anda berasa masuk ke dalam game. Walaupun sekarang ini masih merupakan prototype untuk dunia virtual reality yang sebenarnya, dan belum sempurna pengembangannya.


Fitur secondlife saat saya mengunjunginya

Percepatan Tekhnologi

Tidak kerasa tekhnologi berjalan sangat cepat, rate kecepatannya mengalami percepatan bisa tergolong double exponential, eksponensial ganda, seperti hukum moore untuk transistor dalam microprocessor, artinya bahwa suatu tekhnologi ada perkembangan dari dua tahun sekali, lalu setahun sekali, kini bisa jadi 6 bulan sekali ada update tekhnologinya, atau nanti 1 bulan sehari, dan semakin cepat pada nantinya sehari sekali ada perkembangan yang signifikan..

Masih inget dulu tahun 2007an di kota saya flash disk 128mb mahalnya minta ampun.. Sekarang 1gb atau 16gb udah biasa dimana-mana ada

Belum lagi hardware komputer, cepat sekali kalau saya ingat. Dulu processor dual-core, bahkan quadcore belum ada, dan terus berkembang per tahun. Harganya pun semakin turun dan semakin bisa dijangkau oleh kalangan masyarakat biasa

Jika kita mengalami sendiri dan hidup dalam perkembangan tekhnologi, gak akan terasa, tapi kalau kita melihat tekhnologi dengan mata elang, artinya kita melihat dengan timeline history.

Ingat einstein saat naik sepeda? Kira-kira tahun 1950 an kalau saya tidak salah. Mungkin Einstein pun akan kaget dengan mobil dan tekhnologi transportasi yang ada sekarang.

Bayangkan 50 tahun yang akan datang. Ada tekhnologi baru apa saja yang akan muncul? Virtual reality semacam oculus rift, sudah semakin berkembang, device hanya seperti google glass, tekhnologi hologram mungkin juga sudah ada. Processor komputer mungkin sudah 100 core barangkali (karena efek percepatan tekhnologi).

Jangan sampai nanti kita kaget saat kita tua dengan tekhnologi yang belum kita jangkau secara pikiran.

Tetap terbuka dengan tekhnologi baru. Dan bayangkan apa yang bisa manusia capai dalam 1000 tahun ke depan.
