Jurnal Produktivitas 1, 2, 3, 4

2015-08-01 |

Keluar zona nyaman, adalah salah satu hal yang sangat sulit dilakukan di seluruh peristiwa rentang hidup kita. Menjadi produktif adalah salah satu indikator apakah bisa keluar dari zona nyaman atau tidak.

Ketika saya memutuskan untuk membuat jurnal produktivitas, maka seketika itu juga saya harus mati-matian benar-benar melakukan dan berkomitmen untuk melakukannya.


seekor kucing ingin produktif

Untuk itu saya akan update perjalanan produktif saya, dimana hal ini juga termasuk dalam bagian menjadi produktif –menulis setiap hari adalah salah satu bagian poin penting menjadi produktif.

Pra Pelaksanaan

Karena bulan lalu, ada event tahunan untuk umat muslim, yaitu bulan ramadhan, maka saya harus menghormatinya. Dan karena itu saya tidak melakukan apapun kegiatan yang masuk dalam daftar apa yang akan dilakukan –todolist.

Setelah bulan tersebut usai, saya mulai memikirkan sumber-daya apa saja yang akan saya siapkan.

Sumber-daya disini adalah berupa alat, dalam kasus saya, yaitu aplikasi todo list mobile (handphone). Selain itu artikel yang menunjang dan mengajari saya untuk lebih produktif lagi.

Artikel yang saya temukan selain yang pernah saya bahas sebelumnya di jurnal produktivitas adalah 8 hal yang dilakukan sebelum jam 8 pagi. Di artikel tersebut dijelaskan hal apa saja yang wajib dilakukan sebelum jam 8 pagi.

Beberapa poin saya masukkan ke todolist personal saya. Dan pada hari pertama –nanti akan saya ceritakan, ternyata berhasil meningkatkan produktivitas saya.

Untuk membuat todolist, saya juga perlu sebuah aplikasi atau tool untuk mempermudah melakukan pencatatan task/tugas tiap hari yang harus dilakukan.

Beberapa aplikasi yang bisa digunakan untuk membantu menjadi produktif diantaranya akan saya bahas dibawah ini, tetapi hanya sekilas dan tidak detail:

Any.do

Kita langsung bisa menginstallnya melalui play store (jika android) atau app store (iphone).

Untuk aplikasi ini sebenarnya waktu awal saya sudah menginstallnya:


tampilan any.do
Tampilan any.do termasuk paling minimalis, dan clean. Saya suka UI (User Interface) dari any.do. Tetapi beberapa fitur any.do saya kurang suka, dan beberapa fitur yang disediakan harus membayar premium terlebih dahulu.

Perbandingan fitur bisa dilihat disini.

Walaupun aplikasi ini ada berada di urutan teratas editor’s choice di play store di kategori produktivtas, tetapi pada akhirnya saya uninstall.

Wunderlist

Sejak wunderlist diakuisisi Microsoft, saya rasa tool ini makin powerful, serta cross platform. Terutama untuk windows.

Todo app satu ini sebenernya sangat popular dan dipakai banyak orang khususnya seorang programmer.

Saya sempat mencicipi wunderlist. Tetapi saya kurang suka ui dan ux nya. Diantaranya saya tidak bisa melihat sub task sebagai list.

Tetapi salah satu fitur yang bagus menurut saya adalah adanya public sharing. Jadi setiap orang bisa menerapkan todolist dari orang lain. Dan itu bisa saja sangat membantu.

Karena aplikasi ini cross platform, kita bisa menginstallnya dimana saja. Mulai Android, IPhone, addon untuk Google Chrome, di Windows, dan sebagainya.

Kemungkinan saya juga akan mencobanya lagi ke depan. Sambil membandingkan fitur yang ada.


tampilan wunderlist web

Todoist

Todoist termasuk aplikasi paling atas ketika mencari todo list application di Play Store. Saya memilih aplikasi ini sebagai alat untuk membantu saya mencapai produktif:


Tampilan Todoist desktop, dan beberapa todolist saya

Todoist juga cross platform, saya menginstall aplikasi desktop dan aplikasi mobile nya. Keduanya berguna untuk saya melatih produktivitas.

Sebenarnya kekurangan todoist versi free adalah tidak bisa mengeset reminders (alarm) untuk tiap task. Jadi harus membayar biaya bulanan dan sebagai user premium untuk mengaksesnya.

Tetapi yang saya suka dari todoist adalah tampilannya yang clean dan minimalis. Adanya karma, seperti mengukur seberapa produktif kah kita (seberapa banyak task yang kita selesaikan) dihitung per hari.

Semua aktivitas yang tergolong rutinitas yang saya lakukan saya masukkan ke dalam Personal. Membuat Projects (kategori untuk task kita) pun sangat mudah dan sangat membantu.

Subtask yang bisa dilihat langsung di tampilan list (tanpa mengklik detail, hanya menggunakan tanda panah).Walaupun saya menemukan beberapa bug terkait subtask ini.

Setiap task bisa diset waktu pengerjaannya, apakah berulang, atau satu kali saja. Apakah tiap dua hari sekali, atau tiap satu minggu, atau akhir bulan dst. Juga dalam hal ini todoist menang, karena hanya mengetik kalimat untuk mengesetnya tanpa harus melalui interface yang susah.

Secara keseluruhan saya suka todoist, dan saya memilihnya untuk membantu meningkatkan produktivitas saya.

Aplikasi untuk membantu olahraga

Selain aplikasi todolist, saya juga memasang aplikasi untuk membantu saya berolahraga setiap harinya. Aplikasi 30 day fitness challenge saya pilih untuk menemani saya membantu masalah kesehatan saya.

Hari 1

Karena sebelum hari 1 ini, saya seorang prokrastinator yang ulung. Artinya saya betah tidak melakukan kegiatan apapun selama seharian. Maka hari ke 1 ini merupakan hari transisi yang berat.

Malam sebelum hari ke 1 ini saya tidur tidak sampai 8 jam sehari. Padahal di beberapa artikel menganjurkan bahwa agar produktivitas kita meningkat dan lancar, paling tidak kita harus tidur 8 jam sehari.

Setelah bangun, saya meminum air putih sebanyak 2 gelas. Lumayan enteng. Dan melakukan ritual pagi, yaitu terdiri dari menghirup nafas dalam-dalam selama 5 detik, dan mengeluarkannya selama 10 detik. Setelah itu bersyukur masih hidup dan sehat, dan terakhir bermeditasi.

Karena meditasi adalah hal yang penting, beberapa aplikasi untuk membantu meditasi banyak di play store. Saya menggunakan ini di hari 1.

Meditasi berarti mengosongkan pikiran. Ketika kita diam tidak melakukan apa-apa dan mata terpejam, pasti ada suara bisik dari pikiran kita. Nah meditasi berguna untuk mendiamkan suara-suara cerewet yang berasal dari pikiran kita.

Itu sangat penting sekali untuk meningkatkan produktivitas.

Setelah itu, olahraga yang dipandu aplikasi. Cukup mudah, karena sebelumnya saya sudah pernah melakukannya.

Makan telur tiga biji sebelum mandi air dingin juga saya lakukan. Tetapi pada hari ke 1 ini karena tidak ada persediaan telur, saya bingung bagaimana mencarinya. Karena rumah dekat dengan penjual sayur, maka masalah konsumsi telur ini bukan masalah berat.

Mandi air dingin sebelum melakukan beberapa kegiatan berat adalah salah satu syarat agar kita menjadi produktif. Tetapi pada awalnya mungkin akan terasa mengagetkan bagi tubuh kita.

Sesudah mandi, saya sempatkan untuk membaca buku 30 menit. Buku yang sedang saya baca adalah Hooked. How to Build Habit-Forming Products ditulis oleh Nir Eyal. Buku ini merupakan referensi saya untuk nantinya membuat produk yang bagus untuk user.

Beberapa kegiatan menulis seperti post yang mengenai Haikudan Pekerjaan dalam dunia tanpa uang. Saya lakukan sebelum kegiatan utama.

Tidak lupa mempelajari kembali English Grammar, membaca artikel bahasa English juga saya lakukan. Medium, dan beberapa artikel yang berasal dari email saya baca semuanya.

Setelah semuanya selesai. Lalu saya masuk ke kegiatan utama. Yaitu coding, melanjutkan project yang sempat tertunda karena prokrastinaisi akut.

Malamnya, saya memasang alarm jam 9 malam. Menjaga supaya lekas tidur agar kualitas tidur saya mencapai 8 jam.

Tetapi pada hari pertama agak sedikit melenceng karena saya tidur melebihi jam 10 malam. Artinya ketika saya bangun pagi, saya belum sampai tidur selama 8 jam.

Hari ke 2

Hari kedua ini, saya bangun tepat waktu karena alarm. Seperti biasa saya melakukan morning ritual. Mengecek semua aktifitas yang sudah saya kerjakan.

Hari ke 2 saya bermeditasi masih menggunakan aplikasi mobile saya.

Untuk konsumsi telur, supaya persediaan telur selalu ada, hari kedua ini saya langsung membeli telur sebanyak 1 kg. Ini menjaga supaya saya tidak menghabiskan waktu untuk membeli telur setiap hari.

Hari kedua saya tidak bisa menyelesaikan tulisan untuk diposting ke blog ini. Tetapi masih bisa menulis haiku ini. Beberapa task terbengkalai untuk urusan reading english.

Hari kedua saya tidak sempat belajar grammar English. Karena fokus saya lebih ke task utama.

Hari kedua saya sepertinya telat tidur lumayan parah. Saya takut hari ketiga tidak lancar karena kekurangan tidur.

Hari ke 3

Hari ketiga saya cukup lancar, semua berjalan sesuai rencana. Rutinitas harian dan morning ritual saya lakukan seperti biasa.

Untuk meditasi saya tidak menggunakan aplikasi mobile saya lagi, melainkan saya menggunakan panduan yang ada di Youtube: guided meditation.

Seperti hari kedua, beberapa task mengenai writing dan reading banyak yang terbengkalai. Karena terlalu fokus ke task utama.

Sebagai gantinya, saya menyelsaikan banyak sekali task terkait pekerjaan saya.

Malamnya, saya tidur lumayan baik. Task membaca semua email saya jalani. Dan saya menambah task untuk rutinitas malam seperti menggosok gigi sebelum tidur. Tetapi tetap tidak tenang karena menumpuk pekerjaan di hari kedua.

Hari ke 4

Hari keempat saya bangun tidak tepat waktu. Hari inilah saya menulis jurnal ini. Karena saya juga memiliki hutang task yang banyak dari hari-hari sebelumnya.

Untuk kegiatan ritual pagi, lumayan lancar tadi pagi. Saya masih menggunakan youtube untuk kegiatan meditasi (kegiatan mengosongkan pikiran)

Dan hari ini, mau tidak mau saya harus melunasinya. Beberapa tulisan haiku yang belum sempat terposting di hari ketiga, karena hanya saya simpan di evernote akhirnya saya publish.

=========

Karena hari ke-empat masih dalam proses saat tulisan ini ditulis. Kelanjutan jurnal akan ditulis di jurnal berikutnya.

Pelajaran yang dapat diambil adalah, tidak menunda task setiap harinya. Lebih baik membuat semuanya beres (get things done) setiap hari, daripada memiliki hutang di hari berikutnya.

Terima kasih
